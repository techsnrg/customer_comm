# Customer Communication

A unified customer journey system for ERPNext showing calls, visits, emails and financials in a single timeline view.

## Features

* Unified Customer Journey Timeline Page
* Integrates Emails, Calls, Meetings, Comments
* Integrates Sales Orders, Invoices, Quotations, Payments
* Custom Visit Log DocType

## Installation

Using bench on your Frappe Cloud site:
```bash
bench get-app customer_communication https://github.com/your-username/customer_communication
bench --site your-site install-app customer_communication
```
